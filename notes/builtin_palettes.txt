When replacing a builtin wrestler with an edit wrestler not all palette values are supported. In an edit wrestler all 34 RGB values can be set independently. This is not the case for builtin wrestlers which have a limited number of predefined RGB values to select from. binedit.py tries to find the closest palette match, but if that isn't good enough it is recommended to change your edited wrestlers to use from the following palettes.

The 34 RGB palettes are broken into 10 parts of size 4, 3, 3, 3, 3, 3, 5, 3, 3, and 4 respectively. In the edit wrestler screen you can see the RGB palette parts that are connected because they will have a white line underneath joining them together.
